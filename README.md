# z4r

## Установка

### Интерактивная (меню, выбор версии, ttyd)

```bash
curl -O https://raw.githubusercontent.com/Koviand/z4r/4/z4r/4/z4r && sh z4r
```

Или без сохранения файла:

```bash
curl -fsSL https://raw.githubusercontent.com/Koviand/z4r/4/z4r/4/z4r | sh
```

### Полностью автоматическая (без вопросов)

```bash
curl -O https://raw.githubusercontent.com/Koviand/z4r/4/z4r/4/z4r_a && sh z4r_a
```

Или без сохранения файла:

```bash
curl -fsSL https://raw.githubusercontent.com/Koviand/z4r/4/z4r/4/z4r_a | sh
```

После установки для вызова меню введите в терминале: `z4r`.
