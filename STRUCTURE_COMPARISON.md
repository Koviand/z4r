## ⚠️ Важное замечание о структуре библиотек

В скрипте `z4r.sh` библиотеки ищутся по пути:
```
$SCRIPT_DIR/zapret/z4r_lib/
```

Но в текущей структуре проекта они находятся в:
```
./lib/
```

**Объяснение:**
- В репозитории `zapret4rocket` библиотеки находятся в `lib/`
- При установке на устройство скрипт ожидает их в `/opt/zapret/z4r_lib/`
- Скрипт проверяет наличие библиотек в `zapret/z4r_lib/` относительно места запуска
- Если библиотеки не найдены, скрипт скачивает внешнюю версию из GitHub

**Статус:** ✅ Это нормально - структура репозитория отличается от структуры установки
